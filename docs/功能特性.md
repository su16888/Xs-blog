# 🎯 Xs-blog 功能特性说明

**版本**: V3.0.0正式版
**更新日期**: 2025-12-17

本文档详细说明 Xs-blog 个人主页系统的所有功能和特性。

---

## 📖 系统概述

**Xs-blog** 是一个现代化的个人主页系统，采用前后端分离架构，提供完整的个人品牌展示、内容管理和社交链接功能。系统基于最新的技术栈构建，具有出色的性能和用户体验。

### 🎯 设计理念

- **现代化设计**: 基于最新技术栈的现代化界面设计
- **功能完整性**: 覆盖个人主页的所有核心需求
- **模块化架构**: 易于扩展和维护的模块化设计
- **安全可靠**: 多层安全防护和权限控制
- **响应式适配**: 完美适配桌面端和移动端设备
- **性能优化**: 优化的前端构建和数据库查询

## ✨ 核心功能模块

### 1. 🏠 前台展示页面

#### 个人资料展示
- **头像管理**: 支持自定义头像上传、裁剪和显示
- **个人简介**: 多行文本介绍，支持Markdown格式
- **位置信息**: 显示用户所在位置
- **联系信息**: 邮箱、电话等联系方式展示

#### 社交链接管理
- **自定义图标**: 支持上传自定义图标或使用URL链接
- **二维码支持**: 支持上传二维码图片，点击显示二维码弹窗而非跳转链接
- **自定义排序**: 拖拽排序功能，自定义显示顺序
- **可见性控制**: 控制链接在前台的显示状态
- **资料卡显示**: 可选择是否在个人资料卡片中显示（最多6个）
- **灵活配置**: 支持平台名称、账号、链接地址等完整信息配置

#### 笔记模块
- **Markdown编辑器**: 完整的Markdown语法支持，实时预览
- **分类标签**: 支持笔记分类和标签管理
- **发布控制**: 草稿、发布、置顶等发布状态管理
- **全文搜索**: 支持标题和内容的全文搜索
- **独立页面展示**:
  - 博客风格独立页面（/note/[id]）
  - 支持窗口展示和页面展示两种模式切换
  - 完美适配黑/白/灰三种主题
  - SEO友好，便于分享
- **笔记列表页面**:
  - 独立的全部笔记列表页面（/notes）
  - 列表式布局，每行一个笔记卡片
  - 支持按分类和标签筛选
  - 实时搜索功能（标题、内容、分类、标签）
  - 分页浏览，优化加载性能
  - 博客模式下自动显示在导航栏
  - 列表支持图片文章、无图文章、单列/多列
- **笔记列表展示控制**:
  - 每个笔记可选择是否在列表中展示
  - 隐藏的笔记不会出现在前台列表中
  - 仍可通过直接URL访问隐藏的笔记
  - 管理员在后台可以看到所有笔记
  - 适合创建私密笔记或草稿笔记
- **智能标签/分类筛选**:
  - 笔记详情页标签/分类可点击跳转筛选
  - 首页笔记卡片标签/分类可点击跳转
  - 后台管理页面标签/分类快速筛选
  - 智能跳转：首页跳转到列表页，列表页内部筛选
- **笔记加密**: 针对单独笔记进行加密（**非**高安全性验证）


#### 网站导航
- **网站收藏**: 收藏和管理常用网站链接
- **分类管理**: 手动分类管理
- **快速访问**: 一键跳转和快捷访问功能
- **排序管理**: 自定义网站显示顺序和优先级
- **导航列表页面**:
  - 独立的导航列表页面（/navigation）
  - 网格式布局展示所有导航站点
  - 支持按分类筛选导航
  - 实时搜索功能（名称、描述、分类）
- **展示类型控制**:
  - **前台展示控制**: 可控制网站导航是否在导航页面前台展示
  - **后台显示控制**: 可控制网站导航是否在后台管理系统中显示
  - **灵活配置**: 支持四种展示模式（全部展示/仅前台/仅后台/完全隐藏）
  - **隐私保护**: 敏感或私人网站链接可以隐藏显示
  - **管理优化**: 后台管理界面支持展示类型筛选和批量操作
  - **场景应用**: 个人工作导航、访客推荐链接、私密收藏等不同需求

#### 服务业务展示

- **详情弹窗**:

  - 支持Markdown格式渲染

#### 文本设置功能
- **页面文本自定义**: 可自定义各页面的标题、描述、浏览器标题

- **实时预览**: 修改后立即生效，支持实时预览效果

## 🛠️ 技术特性

### 后端技术栈
- **框架**: Express.js 4.18 - 轻量级高性能Web框架
- **数据库**: MySQL 5.7+ + Sequelize ORM - 关系型数据库和对象关系映射
- **认证**: JWT + bcryptjs - 无状态认证和密码加密
- **安全**: Helmet, CORS, express-rate-limit - 多层安全防护
- **文件上传**: Multer - 文件上传中间件
- **验证**: express-validator - 数据验证和清理

### 前端技术栈
- **框架**: Next.js 14 (App Router) - React全栈框架
- **语言**: TypeScript 5.3 - 类型安全的JavaScript
- **UI库**: React 18 - 现代化用户界面库
- **样式**: Tailwind CSS 3.3 - 实用优先的CSS框架
- **动画**: Framer Motion 12 - 高性能动画库
- **图标**: Lucide React - 现代化图标库
- **Markdown**: react-markdown + remark-gfm - Markdown渲染

### 部署特性
- **容器化**: 完整的Docker和Docker Compose支持
- **微服务**: 前后端分离架构，独立部署
- **负载均衡**: 支持多实例部署和负载均衡
- **监控**: 内置健康检查和性能监控
- **日志**: 完整的日志系统和错误追踪
