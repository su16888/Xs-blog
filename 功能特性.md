# 🎯 Xs-blog 功能特性说明

本文档详细说明 Xs-blog 个人主页系统的所有功能和特性。

---

## 📖 系统概述

**Xs-blog** 是一个现代化的个人主页系统，采用前后端分离架构，提供完整的个人品牌展示、内容管理和社交链接功能。系统基于最新的技术栈构建，具有出色的性能和用户体验。

### 🎯 设计理念

- **现代化设计**: 基于最新技术栈的现代化界面设计
- **功能完整性**: 覆盖个人主页的所有核心需求
- **模块化架构**: 易于扩展和维护的模块化设计
- **安全可靠**: 多层安全防护和权限控制
- **响应式适配**: 完美适配桌面端和移动端设备
- **性能优化**: 优化的前端构建和数据库查询

## ✨ 核心功能模块

### 1. 🏠 前台展示页面

#### 个人资料展示
- **头像管理**: 支持自定义头像上传、裁剪和显示
- **个人简介**: 多行文本介绍，支持Markdown格式
- **位置信息**: 显示用户所在位置和时区
- **在线状态**: 实时在线状态指示器
- **联系信息**: 邮箱、电话等联系方式展示

#### 社交链接管理
- **多平台支持**: GitHub、Twitter、LinkedIn、Email等主流平台
- **自定义图标**: 支持上传自定义图标或使用URL链接
- **二维码支持**: 支持上传二维码图片，点击显示二维码弹窗而非跳转链接
- **自定义排序**: 拖拽排序功能，自定义显示顺序
- **可见性控制**: 控制链接在前台的显示状态（is_visible字段）
- **资料卡显示**: 可选择是否在个人资料卡片中显示（最多6个）
- **灵活配置**: 支持平台名称、账号、链接地址等完整信息配置

#### 笔记系统
- **Markdown编辑器**: 完整的Markdown语法支持，实时预览
- **分类标签**: 支持笔记分类和标签管理
- **发布控制**: 草稿、发布、置顶等发布状态管理
- **全文搜索**: 支持标题和内容的全文搜索
- **独立页面展示** :
  - 博客风格独立页面（/note/[id]）
  - 支持窗口展示和页面展示两种模式切换
  - 完美适配黑/白/灰三种主题
  - SEO友好，便于分享
- **笔记列表页面** :
  - 独立的全部笔记列表页面（/notes）
  - 列表式布局，每行一个笔记卡片
  - 支持按分类和标签筛选
  - 实时搜索功能（标题、内容、分类、标签）
  - 分页浏览，优化加载性能
  - 博客模式下自动显示在导航栏
- **笔记列表展示控制**:
  - 每个笔记可选择是否在列表中展示
  - 隐藏的笔记不会出现在前台列表中
  - 仍可通过直接URL访问隐藏的笔记
  - 管理员在后台可以看到所有笔记
  - 适合创建私密笔记或草稿笔记
- **智能标签/分类筛选** :
  - 笔记详情页标签/分类可点击跳转筛选
  - 首页笔记卡片标签/分类可点击跳转
  - 支持URL参数筛选（?category=xxx&tag=xxx）
  - 筛选条件持久化，可分享筛选链接
  - 后台管理页面标签/分类快速筛选
  - 智能跳转：首页跳转到列表页，列表页内部筛选
- **版本历史**: 笔记编辑历史记录和版本管理
- **评论系统**: 支持读者评论和互动

#### 网站导航
- **网站收藏**: 收藏和管理常用网站链接
- **智能分类**: 自动分类和手动分类管理
- **图标获取**: 自动获取网站favicon和元数据
- **快速访问**: 一键跳转和快捷访问功能
- **排序管理**: 自定义网站显示顺序和优先级
- **访问统计**: 记录网站点击次数和访问时间
- **导航列表页面** (v2.8新增):
  - 独立的导航列表页面（/navigation）
  - 网格式布局展示所有导航站点
  - 支持按分类筛选导航
  - 实时搜索功能（名称、描述、分类）
  - 点击弹窗显示网站详情
  - 博客模式下自动显示在导航栏

#### 主题系统
- **明暗主题**: 支持亮色和暗色主题一键切换
- **多主题支持**: 个人主页、博客两种主题
- **博客主题**:
  - 专为博客风格设计的主题
  - 自动在导航栏添加"全部笔记"和"导航列表"链接
  - 优化的阅读体验和排版
  - 完美适配笔记和导航列表页面
- **自定义字体**: 支持上传和使用自定义字体文件
- **背景设置**: 自定义背景图片和颜色设置
- **颜色主题**: 多种预设配色方案选择
- **CSS定制**: 支持自定义CSS样式覆盖

### 2. 🔧 后台管理系统

#### 仪表盘
- **系统概览**: 关键数据统计和系统状态显示
- **快速操作**: 常用功能的快捷入口和操作面板
- **智能提醒**: 待办事项定时提醒弹窗
- **性能监控**: 系统性能指标和资源使用情况
- **活动日志**: 用户操作记录和系统事件日志
- **数据统计**: 访问量、用户行为等数据统计

#### 个人资料管理
- **基本信息**: 姓名、邮箱、位置等基础信息编辑
- **头像管理**: 头像上传、裁剪、删除和预览
- **密码安全**: 密码修改、强度验证和安全策略
- **隐私设置**: 信息显示权限和隐私保护设置
- **账户设置**: 账户状态管理和安全设置
- **统一管理**: 社交链接管理整合到个人资料页面，统一UI风格
- **链接列表**: 所有社交链接的列表视图和管理
- **链接添加**: 支持多种社交平台的快速添加
- **链接编辑**: 链接信息、图标、二维码、排序等属性编辑
- **图标上传**: 支持上传自定义图标图片（JPG、PNG、GIF、WEBP、SVG）
- **二维码上传**: 支持上传二维码图片，前台点击显示二维码弹窗
- **可见性控制**: 控制链接是否在前台显示（is_visible字段）
- **资料卡显示**: 可选择是否在个人资料卡片中显示（show_in_profile字段）
- **排序管理**: 上移/下移按钮快速调整显示顺序
- **实时预览**: 图标和二维码实时预览功能

#### 笔记管理
- **笔记列表**: 所有笔记的列表视图和筛选功能
- **编辑器**: 功能丰富的Markdown编辑器
- **内容管理**: 笔记内容、标题、分类等属性管理
- **展示模式切换**:
  - 窗口展示：点击笔记在弹窗中打开（默认模式）
  - 页面展示：点击笔记跳转到独立页面（博客风格）
  - 在后台设置中自由切换展示模式
  - 独立页面完美支持主题切换
- **智能标签选择** :
  - 标签搜索和多选
  - 快速创建新标签
  - 标签使用统计显示
- **标签页整合** :
  - 笔记管理和分类管理整合在同一页面
  - 使用标签页切换，减少页面跳转
  - 统一的操作体验和交互模式
- **分类管理增强**:
  - 完整的笔记分类增删改查功能
  - 支持设置分类名称、描述、图标、颜色、排序
  - 表格形式展示分类列表，清晰直观
  - 颜色预览功能，所见即所得
- **发布管理**: 发布状态、置顶、定时发布等控制
- **批量操作**: 批量发布、删除、移动等操作

#### 网站导航管理
- **网站列表**: 所有收藏网站的列表和筛选
- **网站添加**: 网站信息、分类、图标等添加
- **搜索功能** :
  - 实时搜索网站名称、链接、描述
  - 显示搜索结果数量
  - 无结果时显示友好提示
- **标签页整合**:
  - 导航管理和分类管理整合在同一页面
  - 使用标签页切换，操作更便捷
  - 统一的管理体验
- **分类管理增强**:
  - 导航分类的创建、编辑和删除
  - 支持设置分类图标（emoji）和描述
  - 分类排序管理
  - 前台按分类分组展示导航
- **推荐功能** :
  - 支持标记推荐导航
  - 推荐导航显示在专属推荐区块
  - 后台可控制推荐功能开关
- **排序管理**: 网站显示顺序的拖拽排序
- **批量操作**: 批量删除、分类移动等操作

#### 标签管理
- **标签列表**: 所有标签的可视化管理界面
- **标签CRUD**: 创建、编辑、删除标签操作
- **颜色管理**: 8种预设颜色，支持自定义
- **分类管理**: 标签分类和层级管理
- **使用统计**: 显示每个标签的使用次数
- **智能搜索**: 支持关键词搜索和筛选

#### 分类管理
- **统一分类**: 管理笔记、便签、标签的分类
- **分类CRUD**: 创建、编辑、删除分类操作
- **可视化标识**: 支持颜色、图标设置
- **层级分类**: 支持父子分类关系
- **统计功能**: 显示每个分类下的内容数量

#### 系统设置
- **主题设置**: 系统主题、颜色、字体等样式配置
- **字体管理**: 自定义字体的上传、管理和应用
- **背景设置**: 页面背景图片和颜色设置
- **安全设置**: 密码策略、会话管理等安全配置
- **数据管理**: 数据备份、恢复和清理功能

#### 文件上传管理系统
- **模块化存储**: 按后台模块自动分类存储文件
- **文件夹结构**:
  - `/uploads/settings/` - 系统设置相关文件（favicon、背景图片、自定义字体等）
  - `/uploads/blog-theme/` - 博客主题相关文件（logo、轮播图、导航链接图标等）
  - `/uploads/sites/` - 网站导航相关文件（网站logo等）
  - `/uploads/profile/` - 个人资料相关文件（头像、背景图片、社交链接图标和二维码等）
  - `/uploads/notes/` - 笔记管理相关文件（笔记媒体文件等）
- **文件删除同步**: 后台删除文件时自动删除对应的物理文件
- **兼容性保证**: 原有上传功能完全兼容
- **自动分类**: 系统自动根据上传模块将文件存储到对应子文件夹
- **文件管理**: 支持图片、字体等多种文件类型上传和删除

## 🛠️ 技术特性

### 后端技术栈
- **框架**: Express.js 4.18 - 轻量级高性能Web框架
- **数据库**: MySQL 8.0+ + Sequelize ORM - 关系型数据库和对象关系映射
- **认证**: JWT + bcryptjs - 无状态认证和密码加密
- **安全**: Helmet, CORS, express-rate-limit - 多层安全防护
- **文件上传**: Multer - 文件上传中间件
- **验证**: express-validator - 数据验证和清理

### 前端技术栈
- **框架**: Next.js 14 (App Router) - React全栈框架
- **语言**: TypeScript 5.3 - 类型安全的JavaScript
- **UI库**: React 18 - 现代化用户界面库
- **样式**: Tailwind CSS 3.3 - 实用优先的CSS框架
- **动画**: Framer Motion 12 - 高性能动画库
- **图标**: Lucide React - 现代化图标库
- **Markdown**: react-markdown + remark-gfm - Markdown渲染

### 部署特性
- **容器化**: 完整的Docker和Docker Compose支持
- **微服务**: 前后端分离架构，独立部署
- **负载均衡**: 支持多实例部署和负载均衡
- **监控**: 内置健康检查和性能监控
- **日志**: 完整的日志系统和错误追踪
